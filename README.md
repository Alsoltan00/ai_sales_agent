# Evolution AI Sales Agent 🚀

هذا المشروع عبارة عن مساعد مبيعات ذكي يعمل بالذكاء الاصطناعي عبر تطبيق WhatsApp. 
يستخدم المشروع تقنيات متقدمة لفهم ومساعدة العملاء، والرد عليهم بناءً على توفر المنتجات في قاعدة البيانات.

## الميزات الأساسية ✨
- **الفهم الذكي:** يستخدم `meta-llama/llama-3-8b-instruct` (عبر OpenRouter) لفهم نية العميل ومعالجة الرسائل بذكاء.
- **إدارة المحادثة:** يحتفظ بسجل المحادثة لضمان استمرارية السياق باستخدام إطار عمل `FastAPI`.
- **البحث في المنتجات:** يتصل بقاعدة بيانات `Supabase` للبحث وعرض الأسعار وتوافر المنتجات الحديثة.
- **الردود الصوتية والنصية:** يدعم التحويل من نص إلى كلام باستخدام `Edge-TTS` لتوفير ردود صوتية سلسة للعملاء.
- **لوحة أدمين سريعة:** يوفر واجهة لرفع المنتجات وتحديث قاعدة البيانات مباشرة عن طريق ملفات Excel/CSV.

## التقنيات المستخدمة 🛠️
- Python (FastAPI)
- Supabase (PostgreSQL)
- OpenRouter API (LLMs)
- Edge-TTS (Text to Speech)
- Pandas & OpenPyXL

## التثبيت والتشغيل المحلي 💻
1. قم بنسخ المستودع.
2. قم بتثبيت المكتبات الاعتمادية: `pip install -r requirements.txt`
3. قم بإنشاء ملف `.env` وإضافة المفاتيح الضرورية (`SUPABASE_URL`, `SUPABASE_KEY`, `OPENROUTER_API_KEY`).
4. قم بتشغيل الخادم: `python main.py` أو `uvicorn main:app --reload`.
