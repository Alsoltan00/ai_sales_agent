# AI Sales Agent — وكيل المبيعات الذكي

نظام وكيل مبيعات ذكي متكامل يعمل عبر واتساب وتيليجرام.
يتعامل مع الرسائل الواردة، يبحث في بيانات الشركة، ويرد بذكاء وأناقة.

## الواجهات الرئيسية

| الواجهة | الوصف |
|---|---|
| تسجيل الدخول | يوجّه تلقائياً لـ Admin أو التاجر حسب نوع الحساب |
| Admin | إدارة العملاء والمستخدمين والصلاحيات |
| التاجر | جميع أدوات الإدارة والذكاء الاصطناعي |
| الاستقبال | استقبال الرسائل من Telegram / WhatsApp |
| الإرسال | إرسال الردود عبر نفس القنوات |
| التخطيط | تحديد شخصية الوكيل واللهجة والوصف |
| تدريب الذكاء | ربط النماذج واختبارها وتدريبها على الأعمدة |
| مزامنة البيانات | ربط Supabase / Aiven / Google Sheets |
| عرض البيانات | مراجعة البيانات المزامنة |
| الأرقام المصرّحة | فلتر أمان للأرقام المسموح بالرد عليها |

## خوارزمية الرد الآلي

`
رسالة واردة
    ↓ [فلتر الأرقام المصرّحة]
    ↓ [تحليل الرسالة NLU]
    ↓ [بحث في قاعدة بيانات العميل]
    ↓ [تحقق من ملاءمة النتائج]
    ↓ [إرسال الرد المناسب]
`

## التشغيل

1. نسخ ملف الإعدادات:
   cp config/.env.example .env

2. تعبئة المتغيرات في .env

3. تشغيل schema قاعدة البيانات في Supabase:
   database/schema.sql

4. تثبيت المتطلبات:
   pip install -r requirements.txt

5. تشغيل الخادم:
   uvicorn main:app --host 0.0.0.0 --port 8000 --reload

## هيكل المجلدات

    auth/               — المصادقة وتسجيل الدخول
    admin/              — واجهة الإدارة
    merchant/           — واجهة التاجر وجميع أدواتها
    ai_engine/          — محرك الذكاء الاصطناعي
    database/           — قاعدة البيانات والـ schema
    config/             — الإعدادات
    tests/              — الاختبارات
    docs/               — التوثيق
